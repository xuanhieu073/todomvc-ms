# Todo MVC Application

A modern Todo MVC application built with a decoupled architecture, utilizing .NET 10 Minimal APIs on the backend and Angular 20 on the frontend.

## 🚀 Tech Stack

### Backend
* **.NET 10 Minimal APIs**: Lightweight, high-performance HTTP APIs.
* **Carter**: Thin framework layer to modularize Minimal API routes.
* **MongoDB.Entities**: Clean, repository-free data access for MongoDB.
* **FluentValidation**: Strongly-typed validation for incoming requests.
* **AutoMapper**: Object-to-object mapping between entities and DTOs.

### Frontend
* **Angular 20**: Component-driven architecture using the latest signals, standalone components, and modern hydration features.
